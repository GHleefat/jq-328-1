## 1. 产品概述

色盲友好检查器是一款帮助设计师检查设计稿是否对色盲用户友好的在线工具。通过上传图片或截屏，选择不同的色盲类型，实时模拟色盲用户看到的颜色效果，帮助优化配色方案，提升产品的可访问性。

- **核心价值**：帮助设计师快速验证配色方案的包容性，确保产品对所有用户友好
- **目标用户**：UI/UX设计师、前端开发者、产品经理、可访问性专家

## 2. 核心功能

### 2.1 功能模块

1. **主页面**：图片上传区域、色盲类型选择器、Canvas预览区、分屏对比组件
2. **图片处理模块**：支持本地图片上传、屏幕截取、拖拽上传
3. **色盲模拟模块**：红绿色盲、蓝黄色盲、全色盲等多种类型的颜色矩阵转换
4. **对比查看模块**：分屏对比、滑动分割线、原图/模拟图切换

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 主页面 | 上传区域 | 支持点击上传、拖拽上传、屏幕截取，支持常见图片格式 |
| 主页面 | 色盲类型选择器 | 下拉菜单选择不同色盲类型，包含红绿色盲（Protanopia、Deuteranopia）、蓝黄色盲（Tritanopia）、全色盲（Achromatopsia）等 |
| 主页面 | Canvas预览区 | 实时显示原图和模拟效果，支持放大缩小 |
| 主页面 | 分屏对比组件 | 左右分屏显示原图和模拟图，可拖动分割线调整比例 |
| 主页面 | 操作工具栏 | 重置、下载模拟图、切换显示模式等功能 |

## 3. 核心流程

用户上传图片 → 选择色盲类型 → 系统应用颜色矩阵转换 → 实时显示模拟效果 → 用户可通过分屏对比查看差异 → 下载或重置

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深海蓝（#1a365d）作为主色，强调专业和可信赖感
- **辅助色**：珊瑚橙（#ed8936）作为强调色，用于按钮和交互元素
- **背景**：采用深色主题，深灰背景（#1a202c）配合柔和渐变，减少眼睛疲劳
- **按钮风格**：圆角矩形，带有微妙的悬停动画和阴影效果
- **字体**：使用 Space Grotesk 作为显示字体，IBM Plex Mono 作为等宽字体
- **布局风格**：卡片式布局，清晰的功能分区，优雅的留白
- **视觉效果**：玻璃态（Glassmorphism）效果，微妙的噪点纹理，柔和的发光边缘

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI Elements |
|-----------|-------------|-------------|
| 主页面 | 上传区域 | 虚线边框、拖拽动画、文件格式提示、渐变背景 |
| 主页面 | 选择器 | 自定义下拉菜单、图标+文字、选项说明 |
| 主页面 | Canvas区域 | 双Canvas层叠、网格背景、缩放控制 |
| 主页面 | 分割线 | 可拖动滑块、半透明渐变、悬停高亮 |
| 主页面 | 工具栏 | 图标按钮、工具提示、快捷键提示 |

### 4.3 响应式设计

- **桌面端优先**：1280px以上为最佳体验，完整功能
- **平板适配**：768px-1279px，调整布局为上下结构
- **移动端**：小于768px，简化交互，单列布局，隐藏部分高级功能
- **触摸优化**：增大点击区域，支持手势滑动分割线

### 4.4 交互与动画

- **上传动画**：拖拽时边框发光、文件图标弹跳动画
- **转换动画**：颜色渐变过渡效果，平滑的像素过渡
- **分割线交互**：悬停时放大、拖动时的实时反馈
- **页面加载**：渐入动画、元素错峰出现
